Manyana基于开源框架Mirai进行开发，其前身是于2021年立项的Yucca

这个项目的名字来源于
     <p>佛教中的末那识，它是介于意识(第六识)与阿赖耶识(第八识)之间的第七识，一定程度上你可以认为它是“思考中的主体”</p>
     <p>Alan Watts认为“任何思考想法并经历经验的独⽴的“我”，都是⼀个幻觉。”我们可以认为末那识便是这样一个幻觉，它是当下的感官经验与更本原性的阿赖耶识——一切现象的种子 之间的桥梁。</p>
